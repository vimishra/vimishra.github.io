Check if PyTorch is using the GPU

September 4, 2021 · 1 min · Vikas Mishra

This is one of the first things we need to check whenever we are setting up a new system. The easiest way to check this is to use the below commands.

1
2

# First let's import pytorch
import pytorch

Now let’s check if there are multiple devices.

1
2

# How many GPUs are there?
print(torch.cuda.device_count())

Check which GPU is the current one.

1
2

# Which GPU Is The Current GPU?
print(torch.cuda.current_device())

What is the name of the current GPU?

1
2

# Get the name of the current GPU
print(torch.cuda.get_device_name(torch.cuda.current_device()))

Finally check to see if PyTorch is using the GPU.

1
2

# Is PyTorch using a GPU?
print(torch.cuda.is_available())
