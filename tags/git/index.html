Remove Submodule From Git

In order to remove a submodule from git, you need to follow the below 4 steps. Delete the relevant line from the .gitmodules file. Delete the relevant section from .git/config. Run git rm --cached path_to_submodule (no trailing slash). Commit and delete the now untracked submodule files.
